# 🎨 Colorful Calculator

A beautiful, fully functional calculator with colorful gradient buttons and an animated mathematical symbols background!

## ✨ Features

✅ **Colorful Gradient Buttons** - Each button type has a unique vibrant color gradient
✅ **Animated Math Symbols Background** - Mathematical symbols (±×÷=√π∑∫∞ etc.) float across the background
✅ **Smooth Interactions** - Hover effects, ripple animation on click, smooth transitions
✅ **Full Calculator Functions** - Addition, subtraction, multiplication, division, decimal support
✅ **Keyboard Support** - Use your keyboard for faster calculations
✅ **Responsive Design** - Works beautifully on mobile and desktop
✅ **Modern UI** - Glass morphism effect, gradient backgrounds, smooth shadows

## 🎮 How to Use

1. **Click Buttons** or **Use Keyboard**:
   - Numbers: `0-9`
   - Operations: `+ - * /`
   - Decimal: `.`
   - Calculate: `Enter` or `=`
   - Delete Last: `Backspace`
   - Clear All: `Escape`

2. **Example Calculations**:
   - `5 + 3 =` → `8`
   - `10 * 2.5 =` → `25`
   - `100 / 4 =` → `25`
